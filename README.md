# ai-health-assisant
AI Powered Healthcare Management System
📌 Project Overview

Sanjeewani is a full-stack AI-powered healthcare management web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
The platform helps users manage healthcare services digitally through features like doctor appointments, AI diet generation, vaccination tracking, health awareness videos, and admin analytics.

The project aims to improve healthcare accessibility and promote preventive healthcare through smart digital solutions.
